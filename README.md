# Comp
## Jordan Katz
### Description
A repository for my competitive programming workspace
### Dependencies
* GCC
### Installation
1. clone repository
```bash
git clone https://github.com/jordanbkatz/comp.git
```
2. go to directory
```bash
cd comp
```
3. change script permissions to executable
```bash
chmod +x sol.sh
```
### Usage
write solution in [sol.cpp](/sol.cpp) and then run it
```bash
./sol.sh
```
